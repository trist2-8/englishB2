# English B2 Hub

Website tĩnh cho GitHub Pages, chia thành nhiều trang:
- index.html
- resources.html
- vocabulary.html
- grammar.html
- listening.html
- reading.html
- speaking.html
- writing.html
- tracker.html
- assets/styles.css
- assets/app.js

## Cách đưa lên GitHub Pages
1. Mở repo GitHub Pages của bạn.
2. Upload toàn bộ file trong thư mục này lên branch đang publish.
3. Đảm bảo `index.html` nằm ở thư mục gốc của repo.
4. Commit và push.
5. Đợi GitHub Pages cập nhật.

## Ghi chú
- Các checkbox và sổ từ vựng dùng `localStorage`, nên dữ liệu được lưu trên trình duyệt của bạn.
- Các tài nguyên học tập là liên kết ra ngoài tới những nguồn bạn có thể dùng lâu dài.
